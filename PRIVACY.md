# LanaTax Privacy

LanaTax is a local-first browser tool for Solana tax exports.

## What LanaTax Stores

LanaTax stores the wallet address, Helius API key, selected tax year, and cached transaction results in your browser's local storage so the tool can resume your work locally.

## Network Requests

When you request a fetch, LanaTax sends the wallet address and API key to Helius so Helius can return Solana transaction history. SaneApps does not run a server in the middle for these lookups.

## What SaneApps Does Not Collect

LanaTax does not require an account, and SaneApps does not collect your wallet address, API key, transaction history, or exported CSV files from product usage.

## Clearing Local Data

Use the app controls or your browser's site-data settings to clear local storage for `tax.saneapps.com`.
